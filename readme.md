# github_clonner
A github clonner for all

## use
install php and run this command to terminal

```php
php index.php
```

and add repository url to code

### example:

```php
$repositories = [
    "https://github.com/user/repo1",
    "https://github.com/user/repo2",
    "https://github.com/user/repo3"
];
```
